# E-Commerce Test Automation - Project Overview

## Purpose
Automated end-to-end testing for SauceDemo e-commerce application

## Tech Stack
- **Framework:** Playwright v1.40+
- **Language:** TypeScript 5.0+
- **Pattern:** Page Object Model
- **CI/CD:** GitHub Actions
- **Reporting:** Playwright HTML Reporter

## Test Coverage
- **Total Tests:** 15
- **Modules:** 5 (Auth, Products, Cart, Checkout, E2E)
- **Execution Time:** ~35 seconds (parallel)
- **Pass Rate:** 100%

## Key Features
✅ Cross-browser testing (Chrome, Firefox, Safari)
✅ Parallel execution (4 workers)
✅ Auto-retry on failure (2 retries)
✅ Screenshot/video on failure
✅ CI/CD integration
✅ HTML reporting with traces
